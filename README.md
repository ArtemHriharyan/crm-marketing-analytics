# CRM Marketing Analytics

## Project Overview

This project combines CRM and Yandex Metrica data to evaluate marketing channel performance and business metrics.

## Business Problem

CRM data contained completed orders but did not include acquisition source information.

To estimate channel-level performance, a rule-based attribution approach was developed using Yandex Metrica conversion shares.

## Tools

- Python
- PostgreSQL
- SQL
- Yandex Metrica API
- Power BI

## Metrics

- Orders
- Revenue
- Average Order Value
- Average Rental Duration
- Profit per Order
- Customer Acquisition Cost

## Workflow

1. Extract data from Yandex Metrica API
2. Load data into PostgreSQL
3. Transform data using SQL
4. Apply attribution logic
5. Build Power BI dashboards

## Key Findings

- Identified the most profitable marketing channels
- Estimated channel contribution despite missing CRM attribution
- Compared acquisition cost and profitability
